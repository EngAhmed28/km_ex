# King of Muscles - Frontend

Frontend application للموقع الإلكتروني King of Muscles

## المتطلبات

- Node.js (v16 أو أحدث)
- npm أو yarn

## التثبيت

```bash
npm install
```

## التشغيل

```bash
npm run dev
```

التطبيق سيعمل على `http://localhost:5173`

## البناء للإنتاج

```bash
npm run build
```

## الاتصال بالباك إند

تأكد من أن الباك إند يعمل على `http://localhost:5000` قبل تشغيل الفرونت إند.

يمكنك تحديث `AuthContext.tsx` للاتصال بالباك إند API بدلاً من localStorage.
